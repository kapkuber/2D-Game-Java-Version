## Introduction

AdventureQuest 2D is a classic-style 2D game designed for players who enjoy exploration, challenges, and completing tasks. The game offers a visually engaging world with various maps, obstacles, and interactive elements.

## Features

- **Character Movement:** Control your character using the W, A, S, and D keys.
- **Dynamic 2D Maps:** Explore diverse 2D maps filled with challenges and tasks.
- **Task Completion:** Engage in tasks to progress through the game.
- **Interactive Elements:** Encounter NPCs, objects, and obstacles that enhance the gaming experience.
- **Scoring System:** Earn points for completing tasks and overcoming challenges.

## Getting Started

### Prerequisites

- Java Development Kit (JDK) installed
- Java IDE (e.g., IntelliJ, Eclipse)
- LibGDX library

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/AdventureQuest2D.git
   ```

2. Open the project in your Java IDE.

3. Set up the LibGDX library in your project.

## How to Play

1. Run the game from your IDE.

2. Use the following keys to control your character:
   - **W:** Move Up
   - **A:** Move Left
   - **S:** Move Down
   - **D:** Move Right

3. Explore the maps, interact with NPCs and objects, and complete tasks to progress.

## Game Mechanics

AdventureQuest 2D incorporates the following game mechanics:

- **Character Movement:** Use the keys to navigate the character through the map.
- **Tasks:** Complete tasks assigned within the game to advance to the next level.
- **Obstacles:** Overcome obstacles and challenges strategically.
- **Score:** Earn points for task completion and overall progress.
